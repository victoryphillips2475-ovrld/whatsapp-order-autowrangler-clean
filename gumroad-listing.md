### PRODUCT NAME
WhatsApp Order Auto‑Wrangler

### TAGLINE
Turn WhatsApp messages into structured orders for African micro‑retailers.

### DESCRIPTION
Many micro‑retailers in Africa receive orders via WhatsApp and manually copy them into spreadsheets, losing time and making mistakes.

WhatsApp Order Auto‑Wrangler parses those messages, stores them in Appwrite, and provides a web dashboard and a Capacitor React mobile app so merchants can view, confirm, fulfill, and export orders. It includes QR‑code login using Baileys, automatic confirmation messages, and optional payment‑link generation.

You receive the full source repository, Docker‑Compose files, mobile app code, detailed documentation, and a step‑by‑step setup guide. Deploy to Coolify or any VPS in under an hour.

This product is not for large enterprises, SaaS‑only businesses, or anyone needing built‑in payment processing or multi‑language support. It requires self‑hosting and basic DevOps knowledge.

### FEATURES
- WhatsApp order parsing with rule‑based extraction.
- Baileys QR code login for non‑Business WhatsApp users.
- Automatic order confirmation via WhatsApp.
- Merchant dashboard with CSV export and status filters.
- Capacitor React mobile app with QR scanner.
- JWT authentication for secure API access.
- Prometheus metrics and health probes.
- Docker‑Compose deployment with Nginx reverse proxy.
